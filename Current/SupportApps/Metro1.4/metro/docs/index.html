Metro 1.4 FCS is a Web Services framework that provides tools and infrastructure to develop Web Services solutions for the end users and middleware developers. With Metro, clients and web services have a big advantage: the platform independence of the Java programming language.

The distribution contains the following jar files:

webservices-extra-api.jar 		"javax.*" JCP APIs Yes
webservices-rt.jar

webservices-tools.jar Development time tools No
